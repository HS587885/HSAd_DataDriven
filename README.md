## [프로젝트 내용] 
1) 신뢰도가 중요하게 작용하는 제품군인 비타민에 경우 TV 광고에 집중하는 것이 바람직하다고 주장
2) MZ세대가 새로운 수요층으로 등장함에 따라 MZ세대를 타겟으로 하기 위한 광고 전략을 제안
3) 여성이 좋아하는 장르 선호도와 3040이 좋아하는 장르 선호도를 바탕으로 3040세대와 여성이 좋아하는 채널을 뽑아 광고를 넣 자고 제안
